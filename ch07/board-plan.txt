# simple board

## 게시판 프로젝트 데이터 흐름

- 뷰 : 웹페이지
- 컨트롤러 : express-router
- 서비스 : 서비스 function or class
- 데이터 모델 : 데이터모델
- 데이터 엑세스 : 몽고디비모델

1. <뷰> 역할을 하는 웹페이지 -> 익스프레스로 req보냄
2. <컨트롤러>  req받은 것 인증, 유효성 검사 
  if pass => 비즈니스 로직 처리하는 서비스 계층에 있는 함수 호출
3. <서비스> 에서는 컨트롤러에서 받은 데이터로 비즈니스 로직을 처리하거나 데이터 엑세스로 넘기는 모델 객체 생성
4. <데이터 엑세스> 에서는 서비스 계층에서 받은 데이터를 CRUD 수행
5. 데이터 처리 결과를 다시 <서비스>로 넘김
6. <서비스>는 받은 데이터 결과를 <컨트롤러>로 넘김 이때 <뷰>에 필요한 데이터로 가공하기도..
7. <컨트롤러>에서는 <서비스>에 받은 데이터에 실행 결과에 대한 코드 등의 데이터를 웹페이지에 res 로 보냄
8. <뷰>에서는 웹페이지에서 받은 응답에 있는 값을 확인하고 웹페이지를 갱신
